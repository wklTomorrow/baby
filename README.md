# 成长时光盒 - 微信小程序

一款温馨的宝宝成长记录小程序，帮助父母记录和珍藏宝宝生活中的每一个珍贵瞬间。

## 项目简介

成长时光盒是一款专为记录宝宝成长历程而设计的微信小程序，支持照片、文字、视频多种记录形式，通过日历视图直观展示宝宝的成长轨迹。

## 项目接口

### 1. 登录相关

#### 1.1 login
- **功能**：获取用户的OpenID等信息
- **参数**：无
- **返回值**：
  ```json
  {
    "event": {},
    "openid": "用户OpenID",
    "appid": "小程序AppID",
    "unionid": "用户UnionID（如果有）"
  }
  ```

### 2. 数据管理

#### 2.1 initDB
- **功能**：初始化数据库集合和索引
- **参数**：无
- **返回值**：
  ```json
  {
    "success": true/false,
    "message": "操作结果消息",
    "error": "错误信息（如果有）"
  }
  ```

### 3. ID生成

#### 3.1 generateBabyId
- **功能**：生成唯一的宝宝ID
- **参数**：无
- **返回值**：
  ```json
  {
    "success": true,
    "babyId": "生成的唯一宝宝ID"
  }
  ```

#### 3.2 generateRecordId
- **功能**：生成唯一的记录ID
- **参数**：无
- **返回值**：
  ```json
  {
    "success": true,
    "recordId": "生成的唯一记录ID"
  }
  ```

### 4. 宝宝管理

#### 4.1 getAllBabies
- **功能**：获取宝宝列表或根据关键词搜索宝宝
- **参数**：
  - `keyword`（可选）：搜索关键词，用于搜索宝宝昵称
  - `babyId`（可选）：特定宝宝的ID，用于精确查询
- **返回值**：
  ```json
  {
    "success": true/false,
    "babies": [宝宝对象数组],
    "count": 宝宝数量,
    "error": "错误信息（如果有）"
  }
  ```

### 5. 记录管理

#### 5.1 getBabyRecords
- **功能**：获取指定宝宝的所有记录
- **参数**：
  - `babyId`：宝宝ID（必需）
  - `filter`（可选）：筛选条件，可选值有'全部'、'喂养'、'睡眠'、'排便'、'成长'、'其他'
- **返回值**：
  ```json
  {
    "code": 200/400/500,
    "message": "操作结果消息",
    "data": [记录对象数组],
    "error": "错误信息（如果有）"
  }
  ```

#### 5.2 getRecordDetail
- **功能**：获取指定记录的详细信息
- **参数**：
  - `recordId`：记录ID（必需）
- **返回值**：
  ```json
  {
    "code": 200/400/500,
    "message": "操作结果消息",
    "data": [记录对象数组],
    "error": "错误信息（如果有）"
  }
  ```

## 数据库结构

### users 集合
- 存储用户基本信息
- 关键字段：_openid, nickName, avatarUrl, createdAt

### babies 集合
- 存储宝宝信息
- 关键字段：babyId, nickname, birthDate, avatar, _openid, createdAt

### records 集合
- 存储宝宝成长记录
- 关键字段：id, babyId, date, time, type, content, photos, video, _openid, createdAt, updatedAt

### follows 集合
- 存储关注关系
- 关键字段：babyId, babyName, _openid, createdAt
