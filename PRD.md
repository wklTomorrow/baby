# 成长时光盒 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
成长时光盒是一款专为记录宝宝成长历程而设计的微信小程序，帮助父母记录和珍藏宝宝生活中的每一个珍贵瞬间。

### 1.2 产品目标
- 提供便捷的每日记录功能，支持照片、文字、视频多种形式
- 通过日历视图直观展示宝宝的成长轨迹
- 打造温馨、易用的用户体验

### 1.3 目标用户
- 新手父母
- 有记录宝宝成长需求的家庭
- 希望保存珍贵回忆的用户

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 每日记录功能
**功能描述：** 用户可以每天记录宝宝的生活信息

**功能点：**
- **照片记录**
  - 支持从相册选择照片
  - 支持拍照功能
  - 支持多张照片上传（建议最多9张）
  - 支持照片预览、删除
  - 照片自动压缩优化

- **文字记录**
  - 支持文字输入（建议最多500字）
  - 支持表情输入
  - 支持记录标题（可选）
  - 支持标签分类（如：第一次、里程碑、日常等）

- **视频记录**
  - 支持从相册选择视频
  - 支持拍摄视频（建议最长60秒）
  - 支持视频预览
  - 视频自动压缩

- **记录保存**
  - 每条记录自动关联日期
  - 支持编辑已保存的记录
  - 支持删除记录
  - 数据本地存储 + 云端同步（可选）

#### 2.1.2 日历查看功能
**功能描述：** 通过日历视图查看不同日期的记录

**功能点：**
- **日历展示**
  - 月视图日历展示
  - 有记录的日期显示标记点（如小圆点）
  - 支持月份切换（上一月/下一月）
  - 支持快速跳转到今天

- **日期详情**
  - 点击日期查看该日期的所有记录
  - 支持查看记录详情（照片、文字、视频）
  - 支持在详情页编辑/删除记录
  - 支持左右滑动切换不同日期的记录

- **记录列表**
  - 按时间倒序展示记录
  - 支持筛选（按类型：全部/照片/视频/文字）
  - 支持搜索功能（按文字内容搜索）

### 2.2 辅助功能

#### 2.2.1 个人中心
- 宝宝信息设置（昵称、生日、头像）
- 数据统计（记录天数、照片数量、视频数量）
- 设置（隐私设置、数据备份、关于我们）

#### 2.2.2 分享功能
- 支持将单条记录分享给微信好友
- 支持生成精美的记录卡片分享

#### 2.2.3 数据管理
- 数据导出（导出为图片/PDF）
- 数据备份到云端
- 数据恢复

## 3. 页面结构

### 3.1 页面列表
1. **首页（日历页）**
   - 日历视图
   - 今日记录入口
   - 快速添加按钮

2. **记录详情页**
   - 记录内容展示
   - 编辑/删除操作
   - 分享功能

3. **添加/编辑记录页**
   - 照片上传区域
   - 视频上传区域
   - 文字输入框
   - 保存按钮

4. **记录列表页**
   - 时间轴展示
   - 筛选功能
   - 搜索功能

5. **个人中心页**
   - 宝宝信息
   - 数据统计
   - 设置入口

## 4. 技术需求

### 4.1 平台要求
- 微信小程序平台
- 支持微信登录
- 使用微信云开发（可选）或自建后端

### 4.2 性能要求
- 页面加载时间 < 2秒
- 图片压缩后单张 < 2MB
- 视频压缩后单个 < 10MB
- 支持离线查看已加载内容

### 4.3 存储要求
- 本地存储：使用微信小程序本地存储
- 云端存储：使用微信云存储或OSS（用于照片/视频）

## 5. 用户体验要求

### 5.1 设计风格
- 温馨、可爱、简洁
- 色彩以暖色调为主（粉色、蓝色、黄色等）
- 图标和插画风格统一

### 5.2 交互要求
- 操作流程简单直观
- 重要操作有确认提示
- 加载状态有明确反馈
- 错误提示友好清晰

### 5.3 无障碍要求
- 支持字体大小调整
- 按钮大小符合点击规范（最小44x44px）

## 6. 数据统计需求

### 6.1 用户数据
- 记录总数
- 记录天数
- 照片总数
- 视频总数
- 连续记录天数

### 6.2 展示方式
- 在个人中心以卡片形式展示
- 支持生成年度报告（可选）

## 7. 开发优先级

### Phase 1 - MVP（最小可行产品）
1. 每日记录功能（照片、文字、视频）
2. 日历查看功能
3. 记录详情查看
4. 基础的个人中心

### Phase 2 - 增强功能
1. 记录编辑/删除
2. 分享功能
3. 数据统计
4. 搜索功能

### Phase 3 - 高级功能
1. 数据导出
2. 云端同步
3. 年度报告
4. 多宝宝支持（可选）

## 8. 风险评估

### 8.1 技术风险
- 视频上传和播放的兼容性问题
- 存储空间限制
- 数据丢失风险

### 8.2 解决方案
- 做好视频格式兼容性处理
- 提供数据备份功能
- 定期提醒用户备份数据

## 9. 后续规划

### 9.1 功能扩展
- 成长曲线记录（身高、体重等）
- 里程碑提醒
- 家庭共享功能

### 9.2 运营规划
- 用户引导教程
